# Python Calculator

A simple console calculator with two modes: step-by-step input of numbers or calculation of an entire expression at once. The program handles input errors and calculates the result using Python.

## How to use

1. Run `python calculator.py`
2. Select the mode (1 - step-by-step, 2 - full expression)
3. Enter numbers and operations
4. To exit, press Ctrl+C

Supports basic arithmetic operations: addition, subtraction, multiplication, and division.
