# Currency converter for Python
A simple currency converter for Python that calculates how much currency (USD, EUR, RUB) you can buy with one Belarusian ruble.

# Peculiarities
1) Handling all errors
2) Working with the API of the National Bank of the Republic of Belarus
3) User-friendly console interface

# Getting Started
After cloning, run the following command in the terminal:
pip install requests
python main.py
