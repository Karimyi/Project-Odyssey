# Programming Languages Quiz

A simple console-based quiz application that tests your knowledge about programming languages facts and history.

## Features
- 10 interesting multiple-choice questions about programming languages
- Immediate feedback with correct/incorrect answers
- Detailed explanations with fun facts
- Score tracking with percentage calculation
- Error handling for invalid inputs
- Clean console interface

## Getting Started

### Prerequisites
- Python 3.x
- `questions.json` file with quiz data

### Installation
1. Clone or download the project files
2. Ensure you have the `questions.json` file in the same directory
3. No additional libraries required - uses only Python standard library

### How to Run
Simply run the main script:
```
python quiz.py
```
